# Forge Fitness CRM Email Portfolio

A professional CRM and HTML email development portfolio project designed to simulate the customer communication system of a commercial fitness company.

The project includes responsive email templates, lifecycle customer journeys, personalization strategies, CRM workflows, and marketing automation concepts commonly used by modern fitness businesses.

Technologies:

* HTML
* CSS
* MJML
* Git
* GitHub
