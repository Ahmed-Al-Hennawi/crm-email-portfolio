# CRM Personalisation Examples

This folder contains personalisation examples for the Forge Fitness CRM Email Portfolio.

The purpose is to show how email templates can use customer data to create dynamic and personalised messages inside CRM and marketing automation platforms.

## Personalisation Methods Included

1. Liquid examples
2. AMPscript examples

## Why Personalisation Matters

CRM emails are more effective when they are based on customer data such as:

- First name
- Membership type
- Home gym
- Renewal date
- Last visit date
- Recommended class
- Referral code
- Payment status

Instead of sending the same email to every customer, CRM platforms allow brands to send emails that feel relevant to each member.

## Platform Context

### Liquid

Liquid is commonly used in CRM and marketing platforms such as Braze, Shopify, Customer.io, and other lifecycle messaging tools.

### AMPscript

AMPscript is used in Salesforce Marketing Cloud to personalise email content, handle missing data, and display dynamic CRM fields.

## Portfolio Purpose

These examples are not connected to a live CRM platform. They are included to demonstrate how the Forge Fitness email templates could be adapted for real CRM systems.
